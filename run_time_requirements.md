# Run-time Requirements

## Hardware
- Raspberry Pi Zero 2 W (or equivalent Linux device)
- SD card (8GB+)

## Software
- Apache or Nginx
- PHP 7.4+
- MariaDB 10.x
- Chart.js via CDN
- Modern web browser

## Notes
- System runs on local HTTP server
- Data stored in MariaDB and displayed through PHP
- Reports may use static mock data for demo purposes
